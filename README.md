# Выбор объективов для фотосъёмки

## Общее описание
Система предназначена для помощи пользователям в выборе подходящей оптики для разных видов съёмки: портретной, пейзажной, макро и спортивной. Приложение анализирует требования пользователя и предлагает оптимальные варианты объективов из ассортимента магазина.

## Основной положительный сценарий
1. Пользователь открывает приложение и входит в свою учётную запись (или продолжает как гость).
2. Пользователь выбирает тип съёмки (например, "Портрет").
3. Система предлагает список подходящих объективов с кратким описанием характеристик (фокусное расстояние, светосила, совместимость с камерами).
4. Пользователь может просмотреть подробные характеристики выбранного объектива.
5. Пользователь добавляет объектив в «Избранное» или корзину для дальнейшего выбора.
6. Пользователь повторяет шаги 2–5 для других типов съёмки.
7. В конце пользователь получает подборку рекомендуемых объективов на основе своих предпочтений и истории выбора.

## Альтернативные сценарии
### Пользователь не выбирает тип съёмки
Система выдаёт сообщение: "Выберите хотя бы один тип съёмки, чтобы получить рекомендации."  

### Пользователь вводит некорректные данные при фильтрах
Система не позволяет вводить текст в поля с числовыми параметрами (например, диапазон фокусных расстояний) и подсвечивает поле с ошибкой.

## Пользовательские истории
### Я, как **Пользователь**, хочу входить в систему, чтобы получить доступ к своим предпочтениям и сохранённым подборкам.
#### Основной сценарий
1. Пользователь открывает приложение и видит форму входа.
2. Пользователь вводит свой логин и пароль и нажимает кнопку "Войти".
3. Система проверяет данные и открывает главный экран приложения.
#### Альтернативные сценарии
1. Пользователь вводит неверный логин или пароль.
   Система показывает сообщение об ошибке "Неверный логин или пароль".

### Я, как **Пользователь**, хочу выбрать тип съёмки, чтобы получить подборку подходящих объективов.
#### Основной сценарий
1. Пользователь выбирает тип съёмки на главном экране.
2. Система отображает список подходящих объективов.
3. Пользователь просматривает краткие характеристики каждого объектива.

### Я, как **Пользователь**, хочу просматривать подробные характеристики объектива.
#### Основной сценарий
1. Пользователь нажимает на выбранный объектив в списке.
2. Система открывает экран с полными характеристиками: фокусное расстояние, светосила, совместимость с камерами, примеры фотографий.
3. Пользователь оценивает информацию и принимает решение о добавлении в «Избранное» или корзину.

### Я, как **Пользователь**, хочу добавлять объекты в «Избранное» для сохранения выбора.
#### Основной сценарий
1. На экране объектива пользователь нажимает кнопку "Добавить в Избранное".
2. Система сохраняет объектив в списке «Избранное» пользователя.
3. Пользователь может просмотреть все сохранённые объекты в отдельном разделе.

### Я, как **Пользователь**, хочу использовать фильтры для поиска оптимального объектива.
#### Основной сценарий
1. Пользователь задаёт фильтры (бренд, диапазон фокусных расстояний, светосила, цена).
2. Система обновляет список подходящих объективов согласно выбранным фильтрам.
#### Альтернативные сценарии
1. Пользователь вводит некорректные значения.
   Система подсвечивает поле и не позволяет применить фильтр до исправления.

### Я, как **Пользователь**, хочу видеть рекомендации на основе истории моих выборов.
#### Основной сценарий
1. Пользователь открывает раздел «Рекомендации».
2. Система анализирует прошлые выборы и предлагает оптимальные объективы для текущих задач.
